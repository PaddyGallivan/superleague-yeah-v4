    🎯 Start a New Draft

    Snake draft — everyone sees the pool on their phone. Admin picks on the main screen.

    How draft night works 🎯
    1️⃣ Admin hits Start Draft — sets number of rounds
    2️⃣ App shows whose turn it is (jersey + name) on admin screen
    3️⃣ All coaches browse the available pool on their phones — filter by DEF/MID/RUC/FWD
    4️⃣ Coaches call their pick → admin taps PICK → player gone from pool
    5️⃣ ⭐ Gold coaches see AI picks highlighted 🥇🥈🥉 (but grade penalised for using it)
    6️⃣ Snake order: Rd 1 → 1–16, Rd 2 → 16–1, alternating each round
    7️⃣ Skip or Undo available at any time
    5 dollar penalty!
